Git

  • git init — создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий Git.
  • git clone URL — клонирование репозитория из удаленного хранилища на локальную машину
  • git status — позволяет просмотреть какие изменения были сделаны.
  • git add Название файла — добавляет изменения в отслеживаемые файлы.
  • git commit -m "Название" — создает коммит с названием "Название", который указывает на изменения.
  • git log — выводит список всех внесенных изменений.
  • git push — отправляет внесенные изменения из локального в удаленный репозиторий.
  • git pull — получение изменений из удаленного репозитория на локальную машину.
  • git branch Name — создает ветку в локальном репозитории с названием "Name".
  • git checkout Name — позволяет перейти на ветку с названием "Name".
